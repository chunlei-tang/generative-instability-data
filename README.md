# Data for: Stabilizing Decisions under Structural Instability in Generative Systems

This repository contains representative evidence prompts and model-generated outputs used to illustrate instability patterns in generative decision systems.

## Contents

- data/evidence_packs/  
  Representative evidence bundles used as inputs (e.g., clinical guidelines, RCTs, observational studies).

- data/outputs_sample.json  
  Model-generated decisions across repeated runs, including variations in policy decisions, operational intervals, and confidence levels.

## Description

The data illustrate three forms of instability observed in generative systems:

1. Cross-model divergence  
2. Intra-model variability  
3. Policy–operational inconsistency  

Each evidence pack was evaluated multiple times under controlled prompting conditions to capture variability in outputs.

## Notes

- All outputs were generated using publicly accessible large language models.  
- This dataset is intended to demonstrate instability patterns rather than serve as a benchmark.  
- Additional materials are available from the author upon reasonable request.
